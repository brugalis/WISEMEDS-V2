# 💊 WISEMEDS V2
## Smart IoT Medication Dispensing System

WISEMEDS V2 is an ESP32-based smart medication dispenser designed to help patients and caregivers manage scheduled medication intake.

The system combines:
- ESP32 microcontroller
- Servo dispensing mechanism
- Firebase cloud database
- Remote caregiver dashboard
- Automated medication scheduling


# Features

## Device Features

✅ Automatic medication dispensing  
✅ Scheduled medication time  
✅ Manual dispense button  
✅ Servo control system  
✅ Buzzer notification  
✅ LED status indicator  
✅ NTP internet clock  
✅ LittleFS local storage  


## Cloud Features

✅ Firebase Realtime Database  
✅ Remote caregiver monitoring  
✅ Remote dispense command  
✅ Medication status tracking  
✅ Online device monitoring
